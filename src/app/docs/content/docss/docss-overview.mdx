## Documentation Overview

This documentation site provides comprehensive information about the GradSync project, including its architecture, setup, and usage. The site is built using modern web technologies to ensure a seamless and responsive user experience.

### Technologies Used

- **Next.js**: A React framework that enables server-side rendering and static site generation, providing a fast and optimized user experience.
- **Tailwind CSS**: A utility-first CSS framework that allows for rapid UI development with a consistent design system.
- **DaisyUI**: A plugin for Tailwind CSS that provides a set of pre-designed components, making it easy to build beautiful and functional user interfaces.
- **MDX**: A format that lets you seamlessly write JSX in your Markdown documents, enabling rich interactive content.

## Getting Started

First, install the dependencies:

```bash

pnpm install
```

<br />
### Setting Up Environment Variables

Next, rename the `.env.example` file to `.env` to set up the correct environment variables:

```bash
mv .env.example .env
```

<br />
### Running the Development Server

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
